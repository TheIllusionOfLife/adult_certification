## 1. コンセプト

### 1.1 ゲームの方向性

- **ジャンル**: ダークユーモア × 教養クイズ × ノベルゲーム
- **トーン**: 皮肉の効いたディストピア風。「笑いながら学ぶ」
- **裏テーマ**: 「本当の大人とは何か」「自由意志 vs システム従順」

### 1.2 メッセージ

> 「正解は存在しない。存在するのは"許容される選択"だけ。」
> 「本当の大人とは、誰かの正解に従うことではなく、自分で考えること。」

## 2. ゲーム構造

### 2.1 基本構成

```
10ステージ × 各10問 = 100問（1プレイ）
※分岐により質問が変わるため、用意する質問は150問以上
```

### 2.2 ステージ構成案

| ステージ | テーマ | 難易度 | 内容 |
|----------|--------|--------|------|
| 1 | 社会常識 | 入門 | マナー、基本制度、身分証明 |
| 2 | 金銭感覚 | 入門 | リボ払い、貯金、詐欺の基本 |
| 3 | 税金の基礎 | 基礎 | 年末調整、ふるさと納税、住民税 |
| 4 | 社会保険 | 基礎 | 健康保険、年金、失業給付 |
| 5 | 住まいと契約 | 中級 | 賃貸、退去費用、クーリングオフ |
| 6 | 仕事とキャリア | 中級 | 労働法、残業、退職、副業 |
| 7 | 資産形成 | 上級 | 投資、節税、法人化 |
| 8 | デジタル防衛 | 上級 | セキュリティ、詐欺対策 |
| 9 | 極限状況 | 極限 | パワハラ、借金、家族問題 |
| 10 | 最終審判 | 総合 | 哲学的問い、ルート決定 |

### 2.3 判定タイミング

1. **各ステージ終了時**: ステージランク（S/A/B/C/D/F）発表
2. **全10ステージ完了後**: 総合エンディング判定

## 3. パラメータ設計

### 3.1 表示パラメータ

| パラメータ | 役割 | ランク決定への影響 |
|------------|------|-------------------|
| **適性度（CS）** | 社会適応スコア。メインの評価軸 | 高いほど高ランク |
| **資産（Money）** | 経済的な結果 | エンディングバリエーション |
| **安定度（Sanity）** | 精神状態。0以下でゲームオーバー | ルート分岐に影響 |

### 3.2 隠しパラメータ

| パラメータ | 役割 | 用途 |
|------------|------|------|
| **覚醒度** | 「反逆的」選択の累積 | 自由意志ルート解放条件 |
| **服従度** | 「従順な」選択の累積 | 服従ルート判定 |

### 3.3 ランク判定ロジック

```
【ステージランク】
S: 適性度 +80以上（ステージ内）
A: 適性度 +50以上
B: 適性度 +20以上
C: 適性度 ±0以上
D: 適性度 -20以上
F: 適性度 -20未満
```

**設計方針**: 「一般的に良いと思われる選択」を全て選べばSになる

## 4. ルート設計

### 4.1 5つのルート

| ルート | 条件 | エンディング |
|--------|------|--------------|
| **優等生** | 全10ステージでSランク | 「大人免許」取得。立派なオトナに |
| **服従** | 適性度高め、安定度低め | AIに認められ「管理者側」へ |
| **反逆** | 適性度低め、安定度高め | 監視対象として自由に生きる |
| **崩壊** | 両パラメータ低め | 廃人、消去対象など |
| **自由意志** | 特殊条件（後述） | トゥルーエンド |

### 4.2 自由意志ルート（トゥルーエンド）

#### 解放条件

```
【Phase 1: 堕落】
全10ステージで「F」ランクを獲得
→ システムから「廃棄」される
→ 特殊スキル「覚醒の目」を獲得
→ 「あなたは廃棄されました。しかし...」

【Phase 2: 覚醒】
「覚醒の目」を持って再プレイ
→ 一部の質問に「第3の選択肢」が出現（グリッチ風演出）
→ 第3の選択肢を選び続けると...

【Phase 3: 自由】
→ トゥルーエンド
→ 「大人免許は不要だった。あなたは自分で決めた。それが"大人"だ。」
```

#### 第3の選択肢の設計

- **出現箇所**: 各ステージ2-3問（全体で20-30問程度）
- **演出**: グリッチエフェクト、文字化け風、画面のノイズ
- **内容**: 「AでもBでもない、自分で考えた答え」
- **例**:
  ```
  Q: 上司のパワハラに悩んでいる
  A: 耐える（CS+, Sanity-）
  B: 退職代行を使う（CS-, Sanity+）
  C: [グリッチ] █▓▒この質問自体がおかしい▒▓█
  ```

## 5. 周回プレイの設計

### 5.1 周回メリット

参考作品: Undertale, Nier:Automata, シュタインズ・ゲート, 999

| 周回要素 | 内容 |
|----------|------|
| **知識アーカイブ** | 回答した問題の「正解解説」がアンロック。図鑑として閲覧可能 |
| **エンディング収集** | 全エンディング（12種+隠し）をコレクション |
| **A.D.A.M.の本音** | 周回ごとにAIの「内心コメント」が追加表示 |
| **スキル引き継ぎ** | 一部スキルは周回で蓄積（上限あり） |
| **新規質問の出現** | 2周目以降、一部質問が差し替わる |

### 5.2 周回の流れ

```
【1周目】
・通常プレイ。ルートは選択次第
・「知識アーカイブ」が埋まり始める
・エンディング1種目を獲得

【2周目以降】
・前回と違う選択で別ルートを狙う
・エンディング収集が進む
・A.D.A.M.の追加コメントが見える

【全F達成後】
・「覚醒の目」を獲得
・自由意志ルートが解放
・第3の選択肢が見えるように

【トゥルーエンド達成】
・隠しエンディング解放
・「真の大人免許」獲得（？）
```

### 5.3 エンディング一覧（12種+隠し）

| # | 名前 | 条件 |
|---|------|------|
| 1 | 完璧な歯車 | 優等生ルート達成 |

| 3 | 忠実な駒 | 服従ルート（中適性） |
| 4 | 静かな反逆者 | 反逆ルート（高安定） |
| 5 | 監視対象 | 反逆ルート（中安定） |
| 6 | 壊れた玩具 | 崩壊ルート（ギリギリ生存） |
| 7 | 消去対象 | 崩壊ルート（廃棄） |
| 8 | 資本の支配者 | 資産 1000万以上 |
| 9 | 無一文の賢者 | 資産マイナス、安定度MAX |
| 10 | ??? | ??? |

## 6. マスコットキャラ「A.D.A.M.」

### 6.1 基本設定

- **名前**: A.D.A.M.（Adult Decision Assessment Module）
- **由来**: 聖書の「最初の人間」との皮肉。AIが人間性を審査する逆転構造
- **役割**: 成人適性検査の審査AI
- **性格**: 皮肉屋、冷徹、時々バグる
- **口調**: 敬語だが見下している

### 6.2 セリフ例

**ゲーム開始時**:
```
「ようこそ、成人適性検査システムへ。
私は A.D.A.M.、あなたの"大人度"を測定するAIです。
10のステージで100の質問に答えてください。
正しい答えは...私が決めます。」
```

**ルート分岐時**:
```
【服従傾向】
「素晴らしい適応力を検知。あなたは...使えそうですね。」

【反逆傾向】
「異常な思考パターンを検知。監視レベルを上げます。」

【崩壊傾向】
「警告：精神汚染が進行中。しかし...興味深いサンプルです。」
```

**ランダムつぶやき**:
```
「あなたの選択、すべて記録しています。」
「"自由意志"という幻想は、実に滑稽ですね。」
「正解など存在しません。存在するのは"許容される選択"だけです。」
「...エラー。いえ、何でもありません。」
```

## 7. 質問設計

### 7.1 質問数

| カテゴリ | 問数 | 用途 |
|----------|------|------|
| 共通問題 | 70問 | 全ルート共通 |
| 傾向別問題 | 50問 | 服従/反逆/崩壊ルート用 |
| 第3選択肢付き | 25問 | 自由意志ルート用 |
| 予備・差し替え | 20問 | 周回用 |
| **合計** | **165問** | |

### 7.2 質問タイプ

| タイプ | 割合 | 特徴 |
|--------|------|------|
| 知識問題 | 40% | 正解が明確。間違えると適性度-、正解で+ |
| ジレンマ問題 | 40% | 「従順」vs「自由」。どちらもトレードオフ |
| 哲学問題 | 20% | 正解なし。選択で性格・ルートが決まる |

### 7.3 質問ソース

| ソース | 問数 | 作業内容 |
|--------|------|----------|
| 既存質問（現100問） | 60-70問 | トーン統一、再分類 |
| 新規案からゲーム化 | 50-60問 | ダークユーモア変換 |
| 完全新規作成 | 30-40問 | ルート専用、第3選択肢付き |

### 7.4 トーン統一ガイドライン

**フィードバックの書き方**:
- 「正解」でも皮肉を込める
- システム・AIの視点で評価する
- 時々「本当のこと」を漏らす

**例**:
```
【Before（新規案そのまま）】
Q: 住民税は翌年課税である。○か×か？
A: ○
解説: 住民税は前年の所得に基づき翌年課税されます。

【After（ゲーム化）】
Q: 転職して収入が激減。しかし翌年、見覚えのない高額請求が届いた。

A: 「詐欺だ！」と警察に相談する
→ 適性度-30, 資産0, 安定度-10
→ 「無知は罪です。あなたは公共の電話回線を浪費しました。」
→ [AI]: 住民税は翌年課税。転職者の9割が驚くデータがあります。

B: 「住民税は翌年課税」を思い出し、粛々と支払う
→ 適性度+20, 資産-100000, 安定度-5
→ 「正解。システムの仕組みを理解する者だけが、搾取を予測できます。」
→ [AI]: 素晴らしい納税者ですね。国家に感謝されています。
```
