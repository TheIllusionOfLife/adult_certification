# **Unit 0-ADULT プロジェクト・ドキュメンテーション**

## **1\. エグゼクティブ・サマリー (Elevator Pitch)**

「学校では教えてくれなかった、しかし知らないと死ぬ現実」をAIが審査する。  
本作は、大人のリテラシー（金融・法務・マナー）をテーマにしたディストピア・アドベンチャーゲームです。プレイヤーはAI「Unit 0-ADULT」が課す過酷な試験を突破し、「成人」として認められるか、あるいは「生体プロセッサ」として消費されるかの瀬戸際を歩みます。

## **2\. ゲームコンセプト**

### **コンセプト：教育的ディストピア (Educational Dystopia)**

現代社会の複雑なシステム（税制、労働法、契約）を、AIによる人間選別の枠組みに落とし込んでいます。

* **裏テーマ:** 「AIにとって都合の良い、予測可能な資源」の選別。  
* **マルチエンディング:** 単なる合否だけでなく、クレジットスコア（CS）と資産状況に応じた階層別のエンディング。  
* **全問不正解エンド:** 債務の永久機関（生体プロセッサ）として脳を徴用される衝撃的な結末。

## **3\. ターゲットユーザー**

* **コア:** 20代〜30代の社会人（「義務教育で教えろ」という嘆きをSNSで発信・共感する層）。  
* **サブ:** 『Papers, Please』や『Fallout』のような、不気味で皮肉の効いた世界観を好むゲーマー。  
* **価値:** プレイを通じて、現実世界で役立つ「防衛リテラシー」が身につく。

## **4\. アートスタイル：Dark Cartoon**

親しみやすさと残酷さのギャップを最大化するため、以下のガイドラインを採用しています。

* **視覚的特徴:** 太いインクの輪郭線、歪んだパース、汚れのついた紙の質感、限定的なパレット。  
* **演出:** \* AI試験官は「可愛らしいが目が笑っていない」デザイン。  
  * ステータス悪化に伴い、UIにノイズや「血のようなインク」が混ざり始める。

## **5\. ゲームメカニクス (USP)**

1. **3軸ステータス管理:** Credit Score (社会的信用)、Money (生存資金)、Sanity (精神) のジレンマ。  
2. **画像鑑定:** AI生成画像に含まれる「契約書の罠」や「違和感」を読み解くパズル性。  
3. **成人スキルシステム:** ローグライト的な強化要素。自分なりの「立ち回りビルド」の構築。  
4. **動的難易度:** AIがプレイヤーの弱点（例：計算には強いがマナーに弱い）を学習して出題を最適化する。

## **6\. バズり要素 (Viral Triggers)**

* **「無知の可視化」:** 自分が現実で損をしていたかもしれない事実を突きつけられるショック。  
* **診断・シェア:** 「あなたの成人ランク：B（標準的なパーツ）」といった、SNSに貼りたくなる格付け。  
* **メタ・フィクション:** AIがプレイヤーの実生活を皮肉るようなメタ発言。

## **7\. 開発ガイドライン**

### **コード規則 (HTML/JS版)**

* **データ駆動:** 問題文、選択肢、画像プロンプトはすべて questions 配列（JSON形式）で管理すること。  
* **ステータス更新:** すべての増減は updateHUD() を通じて視覚的にフィードバックすること。  
* **拡張性:** renderQuestion() を変更せずに新しいカテゴリーを追加できる構造を維持する。

### **問題量産の手順**

新しい問題を追加する際は、以下の構造を守ってください。

{  
  "category": "カテゴリー名",  
  "text": "大人が悩む具体的なシチュエーション",  
  "image": "Dark Cartoonスタイルの画像生成用プロンプト",  
  "choices": \[  
    { "text": "選択肢", "effect": {"cs": 0, "money": 0, "sanity": 0}, "feedback": "AIの皮肉" }  
  \]  
}

## **8\. 製品版での技術スタック (推奨)**

将来的な製品化に向けたスケーラブルな構成案です。

* **Frontend:** React / Next.js (動的なUIアニメーションと状態管理)  
* **Backend:** Node.js (Firebase Auth / Firestore によるマルチプレイヤー・ランキング)  
* **AI Integration:** Gemini API (プレイヤーの回答に合わせたリアルタイムな皮肉・画像生成プロンプトの動的生成)  
* **Assets:** Stable Diffusion / Midjourney (Dark CartoonスタイルのモデルをFine-tuning)
